# Goal Progress Tracker Examples

These examples show how a runtime or operator can turn an existing Goal Contract plus current execution state into a resumable `Goal Progress Log`.

Each example includes:

- the existing Goal Contract
- the tracking package inputs
- the resulting `Goal Progress Log`
