# Goal-Driven Examples

These examples show how `goal-driven` turns ambiguous requests into canonical Goal Contracts.

Each example includes:

- the original user request
- the clarification assumptions or notes needed to complete the contract
- a final Goal Contract
- why the contract works
- a common failure mode to avoid
